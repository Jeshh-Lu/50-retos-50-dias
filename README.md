# 50 Retos en 50 Días

¡Bienvenido a mi colección de **50 Retos en 50 Días**! 🚀

Este repositorio contiene 50 proyectos de desarrollo frontend, desarrollados con el objetivo de mejorar mis habilidades en diseño web e interacciones del lado del cliente. Está inspirado en el famoso reto de construir 50 proyectos en 50 días.

## 🛠️ Tecnologías Utilizadas

Cada uno de los retos fue construido utilizando herramientas nativas (Vanilla Web):
- **HTML5:** Para la estructura semántica.
- **CSS3:** Para los estilos, layout (Flexbox/Grid), animaciones y colores vibrantes en su tema oscuro.
- **JavaScript (ES6+):** Para la lógica, interactividad y manipulación del DOM.

## 📂 Estructura del Portal

La página principal funciona como un índice interactivo que se encarga de mostrar todos los retos creados. Utiliza la lista dinámica desde un archivo JSON para generar la tabla de proyectos:

- `index.html`: La estructura principal del portafolio.
- `style.css`: Da vida al diseño principal de la página, implementando estilos modernos, tema oscuro y paletas con buen contraste.
- `script.js`: Carga la información de forma dinámica y la inyecta en el DOM.
- `lista_proyecto.json`: Base de datos local que contiene el nombre y la ruta de cada proyecto.
- **Carpetas Individuales**: Cada proyecto, como `01-Expanding_Card`, cuenta con su propio ecosistema de archivos para funcionar de manera independiente.
